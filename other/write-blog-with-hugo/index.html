使用hugo写博客

Written by Dison on April 18, 2021 in tools, 204 words

本地运行hugo命令

hugo server

或者

hugo server -D

加上-D参数，可以显示草稿状态的文章。

本地浏览器访问 http://localhost:1313/ 即可看到博客。

写文章

通过下面命令新建文章，然后手动打开编辑该文件即可，每次保存文章，浏览器会自动重新渲染。

hugo new tech/MyBlog.md

发布博客

将需要发布的文章的内容里draft: false标识的值改为true，然后运行下面命令

hugo

会生成静态文件到public目录，最后将这些静态博客文件推送到github page或者自己的静态文件服务器上即可。
